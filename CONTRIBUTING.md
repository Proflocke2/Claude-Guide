# Contributing

This is a personal solo project — I maintain and update it myself.

That said, if you spot an error or outdated information, you're welcome to open an **Issue** on GitHub. I'll review it when I can.

## What's useful to report

- Factual errors (wrong prices, benchmarks, API behavior)
- Broken or outdated links
- Anthropic docs that contradict something written here
- Typos in code blocks

## What I'm not looking for

- Style suggestions or rewrites
- New chapters or major additions
- Pull requests (I won't merge external PRs)

## How to report

Open an issue with:
1. Which file and section the error is in
2. What's wrong
3. The correct information and ideally a link to the official source

# Beitragen

Dies ist ein persönliches Solo-Projekt — ich pflege und aktualisiere es selbst.

Falls du aber einen Fehler oder veraltete Informationen findest, kannst du gerne ein **Issue** auf GitHub eröffnen. Ich schaue es mir an, wenn ich Zeit habe.

## Was hilfreich ist

- Faktenfehler (falsche Preise, Benchmarks, API-Verhalten)
- Defekte oder veraltete Links
- Anthropic-Docs, die etwas im Guide widerlegen
- Tippfehler in Code-Blöcken

## Was ich nicht suche

- Stil-Vorschläge oder Umformulierungen
- Neue Kapitel oder größere Ergänzungen
- Pull Requests (externe PRs werde ich nicht mergen)

## Wie du es meldest

Eröffne ein Issue mit:
1. Welche Datei und welcher Abschnitt betroffen ist
2. Was falsch ist
3. Die korrekte Information — idealerweise mit Link zur offiziellen Quelle
